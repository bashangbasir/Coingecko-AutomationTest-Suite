# Coingecko-AutomationTest-Suite
Automation framework for Coingecko API - WEB - Mobile

Pre-Requisites:

- Install JDK 8 (version 1.8.0_321)
- Install IDE (preferred IntelliJ IDEA)

How to run the test : 
1. For API testss: 
  `run testng-api.xml`
  
2. For Web tests: 
  `run testng-web.xml`
  
3. For Mobile tests:
  `run testng-mobile.xml`
